Stack-AVP Prediction Protocol
Tool Information
Tool: Stack-AVP (Antiviral Peptide Predictor)
Access: Public web server
Version: As available at time of access (see manuscript Methods section)
Input Data

Input sequences are provided in:

input_sequences.csv

Format:

One peptide sequence per row
Column name: sequence
Procedure
Access the Stack-AVP web server
Upload the input file (input_sequences.csv) or paste sequences
Select default prediction settings
Run the prediction
Download the output results in CSV format
Output

The prediction results are stored in:

output_predictions.csv

Typical output fields include:

sequence
prediction (AVP / non-AVP)
probability score
Notes on Reproducibility
All predictions were performed using default server parameters
No manual post-processing was applied except filtering based on prediction labels
The provided output file corresponds exactly to the results used in the study
Limitations

This tool is hosted externally and may be updated over time.
Minor variations in prediction scores may occur if the server version changes.
